# HealthcareAI ERP

HealthcareAI ERP is an AI-powered Hospital Management System built using Spring Boot, Thymeleaf, MySQL, Bootstrap, and Maven.

## Features

### Admin Module

* Doctor Management
* Patient Management
* Appointment Management
* Billing Management
* Revenue Tracking

### Doctor Module

* View Appointments
* Approve / Reject Appointments
* Medical Reports
* Doctor Availability
* Income Dashboard

### Patient Module

* Book Appointments
* View Medical Reports
* View Bills
* Payment Gateway
* Invoice Generation

### Billing System

* GST Calculation
* Invoice Generation
* Payment Tracking
* QR Payment Simulation

## Technology Stack

* Java 21
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Bootstrap 5
* Maven
